# Dynamic Raid Events

Dynamic Raid Events is a server-side mod for **SPT 4.x** that makes every raid unique by selecting random gameplay events.

## Current Features

- Weighted random event selection
- Event descriptions
- Modular event framework
- Centralized event registry
- Support for enabling and disabling events

## Planned Features

- Boss Hunt
- Boss Convention
- Loot Surge
- Heavy Fog
- Raider Invasion
- Cultist Night
- Scav Army
- Dynamic weather
- Configurable event chances

## Installation

Copy the mod into:

`SPT/user/mods/`
